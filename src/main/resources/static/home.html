Welcome to My App

Log In Sign Up Enter Your JWT
Usage Steps:
1. Click Sign Up and Enter Details
2. Return, and go to the Log In page
3. Copy the JWT you get after Logging In
4. Return to home, and click Enter Your JWT
5. Enter the token to access the website
6. The secret data will be revealed if the JWT is valid
More about JWT:
Normally, JWT would be stored in local storage, and included in the following request headers by the front end.
Here, only a minimal front end is created so user must manually enter JWT.
In this case the JWT is invalidated after 2 mins of creation.
While the JWT is valid, the user can access the complete website.
